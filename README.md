# SKY PV Monitor

Este repositorio contiene el archivo `SkyPV_Monitor_v14.html` listo para ser publicado en GitHub Pages.

## Cómo visualizarlo en GitHub

1. Sube este repositorio a GitHub.
2. Activa GitHub Pages en la configuración del repositorio.
   - Elige la rama `main` o `master`.
   - Selecciona la carpeta raíz `/` como fuente.
3. Abre la URL de GitHub Pages:
   - `https://<usuario>.github.io/<nombre-del-repo>/`
4. Si necesitas abrir el archivo directamente:
   - `https://<usuario>.github.io/<nombre-del-repo>/SkyPV_Monitor_v14.html`

## Nota importante

- La vista normal de archivos en GitHub (el panel de código del repositorio) no ejecuta JavaScript ni carga completamente archivos HTML de la misma forma que un navegador.
- Para ver la aplicación correctamente, usa GitHub Pages o abre el archivo a través del enlace `Raw` / `raw.githubusercontent.com`.

## Alternativa rápida

Si no quieres activar GitHub Pages, puedes usar un visor de HTML como `htmlpreview.github.io`:

`https://htmlpreview.github.io/?https://raw.githubusercontent.com/<usuario>/<nombre-del-repo>/main/SkyPV_Monitor_v14.html`

Reemplaza `<usuario>` y `<nombre-del-repo>` por tu usuario y el nombre del repositorio.
