# SkyPV Monitor

Sistema de Incidencias Fotovoltaicas - Dashboard interactivo para monitoreo de paneles solares.

## Vista en línea

Este proyecto está alojado en GitHub Pages. Puedes verlo en: [https://tu-usuario.github.io/SkyPV_Monitor/](https://tu-usuario.github.io/SkyPV_Monitor/)

## Características

- Dashboard con KPIs y gráficos
- Tabla de incidencias con filtros
- Interfaz responsiva
- Tecnologías: HTML, CSS, JavaScript, Chart.js

## Instalación local

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/SkyPV_Monitor.git
   ```

2. Abre `index.html` en tu navegador.

## Dependencias

- Chart.js (CDN)
- XLSX (CDN)
- Google Fonts
